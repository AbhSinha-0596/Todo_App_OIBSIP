# Todo_App_OIBSIP
# Todo Web Application 📝

A sleek, responsive, and modern **Todo Web Application** developed as a part of the **Oasis Infobytes Web Development Internship**. This project is built using modern frontend technologies to deliver a fast, lightweight, and highly interactive task management tool with full CRUD functionalities.

---

## 🚀 Features

- **Task Management (CRUD):** Effortlessly create, read, update, and delete tasks.
- **Dynamic Task Status:** Toggle completion status instantly with clean visual transitions.
- **Categorization & Filtering:** Filter tasks seamlessly by *All*, *Active*, and *Completed* states.
- **Persistent Storage:** Integrated browser `localStorage` to ensure tasks are preserved across sessions and page refreshes.
- **Lightning Fast Performance:** Bundled with **Vite** for near-instantaneous Hot Module Replacement (HMR) and optimized builds.
- **Modern & Responsive UI:** Styled with **Tailwind CSS** to provide a beautiful, mobile-first design that adapts smoothly to all screen sizes.

---

## 🛠️ Tech Stack

- **Frontend Library:** [React.js (v18+)](https://react.dev/)
- **Build Tool & Bundler:** [Vite](https://vitejs.dev/)
- **Styling Framework:** [Tailwind CSS](https://tailwindcss.com/)
